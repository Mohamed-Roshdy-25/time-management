# Notify

ToDo app dealing with :
  - local notification.
  - sqlflite.
  - GetX as state management.


## Featuers

  - Add (weekly - daily - specific time) task.
  - delete task.
  - Get tasks by selecting a datetime
  - reminder notification before the task time.
  - dark & light mode.
